Road to a Decision: Car Buying Scorecard

Multi-Criteria Ranking Analysis in Microsoft Excel

Project Overview

This Excel project applies a structured Multi-Criteria Decision Analysis (MCDA) framework to objectively rank 11 candidate vehicles for purchase. Rather than relying on subjective preference, the analysis assigns weighted scores to each vehicle across several performance and value dimensions, enabling a transparent, data-backed final recommendation.

Methodology

Step 1 — Raw Data Collection

Eleven vehicles were evaluated across nine attributes: Horsepower, Weight (lbs), Power-to-Weight Ratio, MPG City, MPG Highway, Fuel Efficiency, Drive Type (AWD/FWD), Rear Legroom, Boot Space, Safety Rating, and Total Price. All data was compiled into a structured raw data table to serve as the foundation for further analysis.

Step 2 — Data Normalization

To allow fair comparison across attributes with different units and scales, all numeric values were normalized to a 0–100 scale using min-max normalization. Each car's score per attribute was expressed as a percentage of the maximum value observed in that category. For Total Price, an inverse normalization was applied so that lower cost yields a higher score — reflecting the buyer's preference for affordability.

Step 3 — Weighted Scoring & Ranking

Four key attributes were selected for the final ranking, each assigned a weight based on buyer priority: Power-to-Weight Ratio (30%), Safety Rating (30%), Fuel Efficiency (20%), and Rear Legroom (20%). Each car's weighted score was calculated by multiplying its normalized score in each category by the corresponding weight, then summing the results. Vehicles were then ranked from highest to lowest total score.

Results

The Nissan Maxima emerged as the top-ranked vehicle (92.24), leading in Power-to-Weight ratio. The Acura RLX placed second (91.82), followed closely by the Honda Accord (91.60), which offered the best all-round balance of efficiency, space, and safety.

Conclusion

This project demonstrates how Excel can transform a complex personal decision into an objective, repeatable process. By combining min-max normalization with a weighted scoring model, raw vehicle data is converted into a clear ranked recommendation than going by intution or guessing. The framework is fully adaptable: weights and criteria can be tuned to match any buyer's unique priorities.
